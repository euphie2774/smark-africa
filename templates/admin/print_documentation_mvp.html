Database Models ({{ stats.total_users }} users)

40+ SQLAlchemy models covering: Users, Products, Orders, Transactions, Shopping Cards, Coins, Events, Raffles, Carriers, POS, BNPL, Market Intelligence, and more.

Key Technical Decisions

  • Sync Gunicorn workers to avoid SSL recursion with gevent
  • HTTP-based Resend API to avoid SDK/gevent conflicts
  • Concurrent web scraping (ThreadPoolExecutor) for fast market intelligence
  • Client-side session cookies with secure SECRET_KEY
  • Case-insensitive login (username and email)
  • Two-factor registration with email verification codes
  • Shopping card numbers with prefix {{ settings.get('shopping_card_prefix', '607845') }}
