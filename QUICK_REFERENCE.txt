================================================================================
TROUBLESHOOTING
================================================================================

  PROBLEM: "CSRF token missing"
  SOLUTION: Add {{ csrf_token() }} to your form

  PROBLEM: Voice not working
  SOLUTION: Use Chrome/Edge/Safari, enable microphone, use HTTPS

  PROBLEM: Can't login after changes
  SOLUTION: Clear browser cookies and try again

  PROBLEM: Module not found error
  SOLUTION: venv\Scripts\python.exe -m pip install -r requirements.txt

  PROBLEM: Database error
  SOLUTION: Run db.create_all() to create AuditLog table
