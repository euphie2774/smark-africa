    ┌──────────────┐

           │
           ├─────────────────────────────────────────┐
           │                                         │
           ▼                                         ▼
    ┌──────────────┐                         ┌──────────────┐
    │  Online      │                         │  POS         │

           │                                         │

           │                                         │

    ┌─────────────────────────────────────────────────────┐
    │  System Calculates Credits                          │

    └──────────────────────┬──────────────────────────────┘
