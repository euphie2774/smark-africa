═══════════════════════════════════════════════════════════════
👤 FOR CUSTOMERS
═══════════════════════════════════════════════════════════════

REGISTER FOR CARD:
1. Log in to your account
2. Click "My Rewards Card" in menu
3. Fill registration form
4. Choose payment:
   - Use 10,000+ loyalty credits (FREE)
   - OR pay KSh 700
5. Receive PIN via SMS
6. Collect physical card from admin

USE CARD AT POS:
1. Present card to cashier
2. Cashier scans barcode on BACK
3. You receive SMS notification
4. Approve payment on your phone
5. Done! Payment processed

═══════════════════════════════════════════════════════════════
💼 FOR CASHIERS
═══════════════════════════════════════════════════════════════

PROCESS PAYMENT:
1. Take customer's card
2. Flip to BACK (barcode is on back)
3. Scan barcode with scanner
4. Customer info displays on screen

OPTION 1 - Traditional (No Phone):
5. Ask customer for PIN
6. Customer enters 4-digit PIN
7. Process payment

OPTION 2 - Mobile Authorization (Secure):
5. Click "Request Mobile Authorization"
6. Customer receives SMS on their phone
7. Wait for customer to approve (max 5 min)
8. Process payment when approved

═══════════════════════════════════════════════════════════════
👨‍💼 FOR ADMINS
═══════════════════════════════════════════════════════════════

PRINT CUSTOMER'S CARD:
1. Go to Admin → Shopping Cards
2. Find customer's card
3. Click "Print" button
4. Print BOTH SIDES:
   - Front: Logo + card details
   - Back: Barcode + terms
5. Use PVC card stock (CR80 size)
6. Deliver to customer

ISSUE CARD DIRECTLY:
1. Admin → Shopping Cards
2. Select customer
3. Enter name for card
4. Enter issue fee (if any)
5. Click "Issue and Print"
6. Print card + PIN slip
7. Give both to customer

DELETE CARD:
1. Find card in list
2. Click trash icon (🗑️)
3. Confirm deletion
4. Card permanently removed

📱 SMS INTEGRATION (Optional)
═══════════════════════════════════════════════════════════════

CURRENT STATUS:
- SMS function exists but is placeholder
- Logs messages instead of sending
- Works without SMS for testing

TO ENABLE LIVE SMS:
1. Sign up: africastalking.com (Kenya) or twilio.com
2. Get API credentials
3. Update send_sms_notification() in main.py
4. Install: pip install africastalking
5. Test with real phone number

CODE EXAMPLE:
import africastalking
africastalking.initialize(username='...', api_key='...')

def send_sms_notification(phone, message):
    sms = africastalking.SMS
    response = sms.send(message, [phone])
    return response

═══════════════════════════════════════════════════════════════
❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Q: SMS not sending?
A: SMS gateway not configured yet. Update send_sms_notification()

Q: Barcode won't scan?
A: Make sure scanning BACK of card, not front

Q: Customer didn't get PIN?
A: Check phone number in user account. SMS gateway needed for live.

Q: Authorization expired?
A: Customer must approve within 5 minutes. Create new request.

Q: Can't approve payment?
A: Customer must be logged in to "My Rewards Card" page
